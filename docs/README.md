# Documentation Kakeibo

Ce dossier rassemble le résultat du brainstorming produit/technique mené avant la
grande phase d'évolution de l'application, et les décisions prises depuis.

- [vision.md](./vision.md) — la méthode Kakeibo manuelle et le principe directeur de l'app
- [decisions.md](./decisions.md) — les décisions produit/techniques actées, avec leur justification (la plus à jour des trois)
- [plan.md](./plan.md) — le plan de travail phasé initial (9 phases, toutes implémentées) ; les évolutions ultérieures sont dans decisions.md

Ces documents reflètent l'état des décisions à la date de rédaction. Si une décision
change en cours de route, ce dossier doit être mis à jour en conséquence.
